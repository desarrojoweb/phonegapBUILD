- ARCHIVOS -

pwd

Muestra el directorio actual de trabajo, el lugar que guarda el promnt

Ejemplo del comando

  • pwd

    Muestra en qué lugar del árbol de directorio nos encontramos

ls

Lista los ficheros contenidos en el directorio actual con código de colores y además permite argumentos que amplían su función

Ejemplos del comando con argumentos

  • ls -a

    muestra los archivos ocultos (todos aquellos que empiezan por un punto)

  • ls -l

    información extra de los ficheros contenidos en el directorio donde nos encontramos. Muy útil para saber qué permisos tienen concedidos los ficheros.

  • ls -la

    Unión de los argumentos. Muestra archivos ocultos y permisos de los archivos contenidos dentro del directorio donde estamos ubicados.

dir

lista los ficheros contenidos en el directorio actual

mkdir

Crea directorios

Ejemplos del comando con argumentos

  • mkdir /home/pepe/documentos/arhivoacrear

    Crea archivo en ruta especificada

cd

Utilidad para navegar por el árbol de directorios. Podemos entrar y salir de un directorio gracias a este comando

Ejemplos del comando con argumentos

  • cd ..

    ir a directorio .. es decir,
    al directorio inmeditamente superior

  • cd .

    ir al directorio actual, en realidad esto no supone ningún cambio pero aprendemos que en todos las carpetas de Linux siempre hay dos directorios aunque éste esté vacio. El fichero dos puntos (inmeditamente superior) y el punto (el mismo directorio)

  • cd ~

    nos dirijimos al directorio home del usuario con el que estemos logueados

  • cd /home/pepe/documentos

    nos dirijimos al directorio documentos del usuario pepe

rm

Elimina ficheros (y directorios)

Ejemplos del comando con argumentos

  • rm /home/pepe/descargas/libro.pdf

    elimina libro.pdf

  • rm -r /home/pepe/descargas/libros/

    Elimina el directorio libros y todo sus contenidos. La -r indica que borre todo el contenido ubicado dentro de la ruta indicada. (Recursivo)

rmdir

Elimina un directorio

Ejemplos del comando con argumentos

  • rmdir /home/pepe/descargas/libros/

    Elimina el directorio libros.

cp

Copia ficheros

Ejemplos del comando con argumentos

  • cp /home/pepe/descargas/libro.pdf /home/pepe/documentos/libros/

    copia libro.pdf dentro del directorio libros ubicado en documentos del usuario pepe

mv

Mover o Renombrar

Ejemplos del comando con argumentos

  • mv /home/pepe/descargas/libro.pdf /home/pepe/documentos/libros/

    mueve libro.pdf dentro del directorio libros ubicado en documentos del usuario pepe

  • mv Istoria Historia

    Cambia el nombre Istoria por Historia

cat

Muestra el contenido de un archivo

Ejemplos del comando con argumentos

  • cat /etc/passwd

    Muestra el contenido del archivo passwd

more

Muestra contenido de un archivo, haciendo pausas por líneas

Ejemplos del comando con argumentos

  • more /etc/passwd

    Muestra contenido de passwd con pausas por cada línea

less

Muestra contenido de un archivo pudiendo hacer scroll en la consola con las teclas de dirección

Ejemplos del comando con argumentos

  • less /etc/passwd

    Muestra passwd y podemos movernos por el archivo arriba y abajo

df

Muestra espacio libre de los discos o las particiones

Ejemplos del comando con argumentos

  • df -h

    Muestra los datos del espacio libre de forma comprensible para los humanos

  • df -m

    muestra espacio libre en megabytes

  • df -k

    Muestra en blokes de 1Kb (por defecto)

  • df -T

    Muestra tipo de sistema de archivos

  • df -i

    Muestra los inodos del sistemas

- USUARIOS -

passwd

Cambia contraseña de usuario

Ejemplos del comando

  • passwd

    Cambiamos la contraseña del usuario con el que estemos logueados

  • sudo passwd pepe

    Cambiamos la contraseña del usuario pepe desde otro usuario logueado

  • sudo passwd -S pepe

    Indica el estado de las contraseñas del usuario pepe. (Bloqueada si/no , última modificación de la clave, etc...)

chmod

Gestiona permisos de un fichero

Ejemplos del comando con argumentos

  • chmod 777 file.txt

    Permisos de lectura, escritura y ejecución para todos los usuarios

  • chmod 666 file.txt

    Permisos de lectura y escritura pero NO ejecución para todos los usuarios

  • chmod 766 file.txt

    Permisos de lectura, escritura y ejecución para el dueño del fichero. Permisos de lectura y escritura para usuarios que permanezcan al mismo grupo y los otros (todos los demás)

  • chmod 733 file.txt

    Acceso total al propietario. Escritura y ejecución al grupo y a los otros

  • chmod 555 file.txt

    Permisos de lectura y ejecución al dueño, grupo y otros

  • chmod +r file.txt

    Permisos de Lectura a todos los usuarios

  • chmod u+w file.txt

    Permisos de escritura al dueño

  • chmod -x file.txt

    Quita, resta, permisos de ejecición a todos los usuarios

chown

Cambia propietario de un fichero (archivo o directorio)

Ejemplos del comando con argumentos

  • chown NuevoPorpietario /home/pepe/documentos/file.pdf

    Cambia de dueño el archivo file.pdf

  • chown -R NuevoPropietario /home/pepe/desacargas/carperta1

    Es recursivo (-R) Cambia de dueño el directorio "carpeta1" y todo el contenido que haya en su interior

sudo

Ejecuta comandos con privilegios de superusuario. (SuperUser do). Requiere contraseña

Ejemplos del comando con argumentos

  • sudo chmod 777 file.txt

    Otorga permisos de ejecución de SuperUsuario al comando chmod

su

Cambio de usuario. Requiere contraseña

Ejemplos del comando con argumentos

  • su pepe

    Permite loguearse con el usuario pepe

  • su -

    Permite loguearse como root

groupadd

Crea un nuevo grupo de usuarios

Ejemplos del comando con argumentos

  • groupadd testers

    Crea grupo "testers"

groupdel

Borra grupo existente

Ejemplos del comando con argumentos

  • groupdel testers

    Borra grupo "testers"

adduser

Crea nuevo usuario. Crea también su carpeta Home. Es un Script programado en Perl

Ejemplos del comando con argumentos

  • adduser juan

    Crea usuario Juan y su carpeta home

useradd

Crea usuario pero no crea su carpeta home. Es un programa binario y funciona en todas las distros.

Ejemplos del comando con argumentos

  • useradd juan

    Crea al usuario juan SIN crear su carpeta Home

  • useradd -m juan

    Crea al usuario juan y ahora sí, (-m) crea su carpeta home

  • useradd -m -g UnGrupo -G OtroGrupo pepe

    Creamos el usuario pepe que pertenecerá a dos grupos: "UnGrupo" y "OtroGrupo"

deluser

Borra usuario y también su carpeta home

Ejemplos del comando con argumentos

  • deluser juan

    Borra usuario juan y su carpeta Home

userdel

Borra usuario SIN borrar su carpeta Home

Ejemplos del comando con argumentos

  • userdel juan

    Borra a juan pero no su Home

- PAQUETES -

dpkg

Instala paquetes .deb existentes en el sistema. No descarga nada

Ejemplos del comando con argumentos

  • dpkg -i man-db_2.7.0.2-5_amd64.deb

    Instala el paquete

  • dpkg --unpack man-db_2.7.0.2-5_amd64.deb

    Desempaqueta paquete sin instalarlo

  • dpkg --configure man-db

    Instala paquete previamente desempaquetado. Esto sirve para depurar posibles errores

  • dkpg -r man-db_2.7.0.2-5_amd64.deb

    Elimina el paquete pero persisten los archivos de configuración. Si más tarde se reinstala el paquete, se conserva la configuración que teníamos antes de desinstalarlo

  • dpkg -P man-db_2.7.0.2-5_amd64.deb

    Se desintala el paquete y todos los archivos de configuración.

apt

Encuentra, instala y actualiza paquetes a través de la red y sus repositorios

Ejemplos del comando con argumentos

  • sudo apt update

    Actualiza la base de datos de paquetes. No podemos instalar un paquete cuya existencia desconoce la máquina

  • sudo apt upgrade

    Actualizamos los paquetes que tenemos instalados en nuestro equipo

  • sudo apt full-upgrade

    Actualización de paquetes instalados en nuestro equipo de manera más intensa y profunda

  • sudo apt install mplayer

    Descarga e instala el paquete mplayer

  • sudo apt install [paquete1] [paquete2] [paquete3]

    Instalación de varios paquetes en una misma línea de comando. (Sin corchetes)

  • sudo apt remove mplayer

    desarrolla tal acciónDesinstala el paquete mplayer pero se conservan los arhivos de configuración del mismo. Si reinstalamos el paquete, mantedríamos la configuración previa

  • sudo apt purge mplayer

    Desinstala el paquete mplayer y borra los archivos de configuración del mismo

  • apt search [unaBusqueda]

    Buscamos algo dentro de los repositorios que tengamos configurados. (Sin los corchetes)

  • apt show mplayer

    vemos el contenido de un paquete, sus dependencias, tamaño, descripciones, etc.

  • sudo apt autoremove

    elimina librerias y paquetes que se instalaron automáticamente para completar la instalación de un paquete que queríamos instalar

Ejemplos del comando con argumentos

  • cd -a

netstat

lsof

Útil para consultar qué ficheros está siendo usados por qué procesos.

kill

Ejemplos del comando con argumentos

    cierra proceso que tenga ese PID

  • kill -1 pidProceso

    fuerza volver a cargar la configuración del proceso con ese PID

free

Ejemplos del comando con argumentos

    Muestra RAM en MB

ps

Ejemplos del comando con argumentos

- OPERADORES -

>

Redirige la salida de un comando a donde le digamos. La salida por defecto es el monitor, sin embargo, podemos indicar que queremos esa salida del comando (lo que se muestra en el monitor) en otro sitio

Ejemplos del comando con argumentos

  • ls > listado

    Con este comando conseguirmos que se escriba un fichero llamado "listado" cuyo contenido sea la salida del comando ls, es decir, el fichero "listado" contendrá un listado del contenido de la carpeta donde se hizo el ls. Especial atención, a que al escribir este operador, SIEMPRE se va a sustituir el contenido del archivo "listado" por la salida correspondiente. Es decir, este operador, borra y escribe la salida.

>>

Redirige la salida de un comando a donde le digamos. La salida por defecto es el monitor, sin embargo, podemos indicar que queremos esa salida del comando (lo que se muestra en el monitor) en otro sitio

Ejemplos del comando con argumentos

  • ls >> listado

    Igual que el anterior solo permanecen los datos anteriores que hubiera en el fichero "listado". No borra; añade.

|

Este operador es conocido por el nombre de tuberías. Lo que conseguimos es que la salida del primer comando sea la entrada del segundo comando. En cierta manera, me lo imagino como un "and". En los ejemplos queda más claro.

Ejemplos del comando con argumentos

  • ls | less

    Con esto conseguimos que se muestre el listado correspondiente a ls y a la vez que se muestre línea a línea gracias a less. Es decir, conseguimos listar el contenido de una carpeta (ls), línea a línea (less)
