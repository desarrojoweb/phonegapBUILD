Comandos Linux - ARCHIVOS -

pwd

Muestra el directorio actual de trabajo, el lugar que guarda el promnt

Ejemplo del comando

  • pwd Muestra en qué lugar del árbol de directorio nos encontramos

ls

Lista los ficheros contenidos en el directorio actual con código de colores y además permite argumentos que amplían su función

Ejemplos del comando con argumentos

  • ls -a muestra los archivos ocultos (todos aquellos que empiezan por un punto)
  • ls -l información extra de los ficheros contenidos en el directorio donde nos encontramos. Muy útil para saber qué permisos tienen concedidos los ficheros.
  • ls -la Unión de los argumentos. Muestra archivos ocultos y permisos de los archivos contenidos dentro del directorio donde estamos ubicados.

dir

lista los ficheros contenidos en el directorio actual

mkdir

Crea directorios

Ejemplos del comando con argumentos

  • mkdir /home/pepe/documentos/arhivoacrear Crea archivo en ruta especificada

cd

Utilidad para navegar por el árbol de directorios. Podemos entrar y salir de un directorio gracias a este comando

Ejemplos del comando con argumentos

  • cd .. ir a directorio .. es decir, al directorio inmeditamente superior
  • cd . ir al directorio actual, en realidad esto no supone ningún cambio pero aprendemos que en todos las carpetas de Linux siempre hay dos directorios aunque éste esté vacio. El fichero dos puntos (inmeditamente superior) y el punto (el mismo directorio)
  • cd ~ nos dirijimos al directorio home del usuario con el que estemos logueados
  • cd /home/pepe/documentos nos dirijimos al directorio documentos del usuario pepe

rm

Elimina ficheros (y directorios)

Ejemplos del comando con argumentos

  • rm /home/pepe/descargas/libro.pdf elimina libro.pdf
  • rm -r /home/pepe/descargas/libros/ Elimina el directorio libros y todo sus contenidos. La -r indica que borre todo el contenido ubicado dentro de la ruta indicada. (Recursivo)

rmdir

Elimina un directorio

Ejemplos del comando con argumentos

  • rmdir /home/pepe/descargas/libros/ Elimina el directorio libros.

cp

Copia ficheros

Ejemplos del comando con argumentos

  • cp /home/pepe/descargas/libro.pdf /home/pepe/documentos/libros/ copia libro.pdf dentro del directorio libros ubicado en documentos del usuario pepe

mv

Mover o Renombrar

Ejemplos del comando con argumentos

  • mv /home/pepe/descargas/libro.pdf /home/pepe/documentos/libros/ mueve libro.pdf dentro del directorio libros ubicado en documentos del usuario pepe
  • mv Istoria Historia Cambia el nombre Istoria por Historia

cat

Muestra el contenido de un archivo

Ejemplos del comando con argumentos

  • cat /etc/passwd Muestra el contenido del archivo passwd

more

Muestra contenido de un archivo, haciendo pausas por líneas

Ejemplos del comando con argumentos

  • more /etc/passwd Muestra contenido de passwd con pausas por cada línea

less

Muestra contenido de un archivo pudiendo hacer scroll en la consola con las teclas de dirección

Ejemplos del comando con argumentos

  • less /etc/passwd Muestra passwd y podemos movernos por el archivo arriba y abajo

df

Muestra espacio libre de los discos o las particiones

Ejemplos del comando con argumentos

  • df -h Muestra los datos del espacio libre de forma comprensible para los humanos
  • df -m muestra espacio libre en megabytes
  • df -k Muestra en blokes de 1Kb (por defecto)
  • df -T Muestra tipo de sistema de archivos
  • df -i Muestra los inodos del sistemas

Comandos Linux - AYUDA y MANUALES -

man

manual completo de un comando

Ejemplos del comando con argumentos

  • man man muestra manual del comando man
  • man ls muestra manual del comando ls
  • man mount muestra manual del comando mount

--help

muestra una ayuda rápida de un comando cualquiera

Ejemplos del comando con argumentos

  • ls --help Muesrtra una ayuda básica del comando ls
  • mount --help Muestra una ayuda básica del comando mount
  • mkdir --help Muestra una ayuda básica del comando mkdir

whatis

Ayuda muy breve de cualquier comando. Información muy reducida

Ejemplos del comando con argumentos

  • whatis mount Muestra para qué sirve mount, pero no cómo se usa.
